## Design Direction
Modern boutique bakery with a clean, elegant aesthetic. The design should feel refined, welcoming, and minimal, appealing to customers who value quality and craftsmanship.

The visual style prioritizes whitespace, soft neutrals, and subtle accents rather than bright or playful colors.

---

## Color Palette

Primary Color (Headings & CTAs):
- #2F2F2F (Charcoal)

Secondary Accent:
- #C9A24D (Soft Gold)

Background:
- #FFFFFF (White)
- #F7F7F7 (Light Gray)

Text:
- #333333 (Primary text)
- #666666 (Secondary text)

Borders & Subtle Dividers:
- #E0E0E0

---

## Logo
The Sweet Crumbs Bakery logo features a centered, balanced layout with a minimalist cupcake icon. The typography-first wordmark reflects a modern boutique patisserie aesthetic, using a refined serif for the brand name and a clean sans-serif for supporting text. Accent elements in soft gold add warmth without overpowering the design.

## Typography

Primary Font (Headings):
- Serif-style font (e.g., Playfair Display or similar)

Secondary Font (Body Text):
- Sans-serif font (e.g., Inter, Lato, or similar)

Typography Guidelines:
- Headings should feel elegant and spaced
- Body text should remain clean and readable
- Consistent line-height and spacing throughout

---

## Layout Guidelines
- Max content width: ~1200px
- Generous padding and margins
- Clear visual hierarchy
- Section-based layout with consistent spacing

---

## UI Components
- Buttons: rounded or soft edges, minimal hover effects
- Navigation: simple horizontal nav (mobile-friendly)
- Cards: used for menu items or features
- Forms: clean inputs with visible focus states

---

## Accessibility Guidelines
- Sufficient color contrast for all text
- Visible focus states for keyboard navigation
- Semantic HTML elements used throughout
- Proper labels for form inputs

---

## Responsive Behavior
- Mobile-first approach
- Navigation adapts for smaller screens
- Content stacks vertically on mobile
- Touch-friendly buttons and links
